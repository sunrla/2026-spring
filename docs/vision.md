# NUVE — 가상 브랜드 비전 문서

> 이 문서는 학교 프로젝트(2026 봄학기 바이브 코딩)에서 사용하는
> 가상 브랜드 NUVE의 정체성·비전·범위를 정의한다.
> 실제 회사와 무관하며, 실제 회사 자산(특허·제품·매장)을 사용하지 않는다.

프로젝트: 2026 봄학기 바이브 코딩 학기 프로젝트

## 1. 브랜드 정체성

### 이름
**NUVE** (전체 대문자)
- 발음: 누베 / noo-vay / ヌーヴェ
- 어원: New + Vera (라틴어 "진실") — "당신만의 새로운 피부 진실"

### 미션 스테이트먼트
> **"당신이 쓰는 화장품은 당신만을 위한 것이어야 합니다."**

### 브랜드 슬로건
- 영어: **"Decoded for you."**
- 한국어: **"오직 당신을 위한"**

### 가치 제안
**"5분의 진단, 평생의 처방"**

---

## 2. 브랜드 톤

### 방향
A-2 정밀 테크 사이언스 (Precision Tech Science)

### 시각 정체성
- 컬러: 다크 네이비 + 실버 + 데이터 시각화 액센트
- 타이포: 모던 산세리프 (시스템 폰트 기반)
- 그래픽: 점수 차트, 분자 구조, 미니멀 라인

### 카피 톤
- 단정적, 약속 중심
- "정확히", "오직", "당신만의" 같은 단정 어휘 선호
- 광고 과장 표현 지양 (의약품 스타일 신뢰 어조)

## 3. 핵심 가치 3가지

### 1) 개인화 (Personal)
- 24타입 같은 유형 분류가 아닌, 점수 벡터 기반의 거의 유일한 결과
- "이 사람만을 위한 처방"

### 2) 정밀 (Precision)
- 피부과학·데이터 기반 진단
- 직관·감성이 아닌 측정·계산 기반

### 3) 속도 (Speed)
- 5분 안에 진단 완료
- 시간 절약을 핵심 가치로 약속

### 기본 정보
- 이름: Maya
- 나이: 36세
- 거주지: 도쿄 (일본 또는 동아시아 부유층 외국인 관광객)

- "시간 없는데 후회하긴 싫다"
- "비싸도 효과 있는 거 사고 싶다"

### 구매 행동
- 한국 여행 시 화장품 예산 30~80만원
- 면세점·백화점 익숙
- "여기서만 살 수 있는" 한정·맞춤·프리미엄에 끌림
- 5분 안에 결정 끝내고 싶지만, 아무거나 사기는 싫음

## 6. 회사 실제 사업과의 차별화

NUVE는 학교 프로젝트의 가상 브랜드로, 실존 회사와 의도적으로 다음과 같이 다르다.

### 차별화 축
1. **타겟**: NUVE = 동아시아 부유층 외국인 관광객, 시술 전 데일리 프리미엄 / 회사 = (회사 정보 미기재)
2. **가격대**: NUVE = 프리미엄 (15~30만원) / 회사 = (회사 정보 미기재)
3. **채널**: NUVE = 디지털 진단 → 가상 매장 예약 / 회사 = (회사 정보 미기재)
4. **알고리즘**: NUVE = 학교 프로젝트만의 독자 점수 벡터 시스템 / 회사 = (회사 정보 미기재)
5. **성분 DB**: NUVE = 일반 공개 정보 기반 학교 큐레이션 / 회사 = (회사 정보 미기재)

### 절대 원칙
- NUVE 설계 시 회사 특허 문서를 참고하지 않는다
- 회사 실제 매장·제품·고객 정보를 NUVE 자산으로 가져오지 않는다
- 회사 실명·매출·매장 위치는 어디에도 기재하지 않는다

## 8. Tier 구조 (구현 범위 계층화)

### Tier 0 — 학교 중간고사 핵심 (완성 우선)
- 홈 진단 (7~10문항)
- 점수 벡터 계산 알고리즘
- 성분 추천 (20~25개 DB 기반)
- 옵션 1 (가상 완제품) / 옵션 2 (조제 예약) 분기
- 처방전 출력
- 이메일 예약 시뮬레이션
- 가상 매장 시나리오

### Tier 1 — 학교 기말 추가
- 라이프스타일 차원 추가
- 환경/기후 차원
- 구독 제안 UI (실제 결제 X)
- 결과 정확도 피드백
- 매장 모드 (오프라인 태블릿용 UI)

### Tier 2 — 회사 버전 영역 (학교 미구현)
- 국가/인종/질병 등 다차원
- 실제 결제 시스템
- 장기 트래킹·데이터 누적
- 실제 매장·예약 통합

### Tier 3 — 회사 한정 (학교 영역 외)
- 실제 조제 기계 연동
- 실제 개인정보 보호법 준수 시스템
- 실제 고객 케어 운영

### 학교 프로젝트는 Tier 0~1까지만 다룬다.

## 10. 비전 (Tier 0 너머)

### NUVE의 장기 비전 (학교 프로젝트 범위 외)
NUVE는 단순 스킨케어 브랜드가 아니라, **사용자의 라이프스타일·환경·건강 데이터를 통합 분석해 전인적 관리를 제공하는 플랫폼**을 지향한다.

확장 가능 차원:
- 피부 데이터 (현재 Tier 0)
- 라이프스타일: 수면, 스트레스, 식단
- 환경: 거주 지역, 기후, 미세먼지, UV
- 신체: 호르몬, 영양 상태, 운동
- 건강: 알레르기, 만성 질환

이 비전은 학교 프로젝트의 Tier 0~1에서 직접 구현하지 않으며, 미래 확장 가능성을 명시하는 용도이다.
